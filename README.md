## Proyecto Final

Diseño de un sistema de **estacionamiento carrusel automatizado** con monitoreo IoT. Incluye arquitectura de red, selección de hardware, protocolo de comunicación y plan de implementación.

Ver documentación en [`Proyecto final/`](./Proyecto%20final/).

---

## Stack tecnológico

- **Hardware:** ESP32, sensores DHT22, actuadores simulados
- **Protocolos:** MQTT, WiFi
- **Plataforma cloud:** ThingsBoard
- **Análisis de datos:** Python, Pandas, Jupyter Notebook
- **Entorno de simulación:** Wokwi
